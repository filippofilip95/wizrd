# Operations Directory

This folder contains business operations - finance, processes, and admin templates.

## Folder Structure

```
operations/
├── templates/               # Operational templates
│   └── weekly-review.md     # Weekly review template (create as needed)
│
├── finance/                 # Financial tracking (create as needed)
│   ├── invoices/            # Invoice records
│   └── expenses/            # Expense tracking
│
├── processes/               # Standard procedures (create as needed)
│
└── weekly-reviews/          # Review logs (create as needed)
```

## Monthly Operations Checklist

```markdown
## End of Month Tasks
- [ ] Generate invoices for all active clients
- [ ] Reconcile expenses
- [ ] Review client statuses
- [ ] Update context.md for all projects
- [ ] Plan next month's content
- [ ] Extract lessons learned from completed projects
```
