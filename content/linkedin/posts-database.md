## Database Structure

Each post entry includes:
- **Date**: Publication date
- **Pillar**: Which content pillar (e.g., CUT THE HYPE / 1-WEEK WIN / BUILDER'S CUT)
- **Format**: Text / Image / Video / Carousel
- **Status**: Published / Ready / Draft
- **Content**: Full post text
- **Hashtags**: Used hashtags
- **Engagement**: Likes, comments, impressions
- **Why This Works**: Analysis of effectiveness
- **Source**: What inspired this post (project, lesson, insight)
